We often call a system stable when it only resists the perturbations we know how to apply.
That is not stability; it is familiarity.
If the boundary is porous to variables we are not measuring, the failure has already entered the model.
A better fragment asks which hidden exchange would invert the conclusion.
If the answer is obvious in hindsight, the assay was underdesigned.
